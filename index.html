About the App

Crew Rest Calculator is a professional tool designed specifically for pilots and cabin crew to efficiently divide rest periods equally during flights. It ensures fair distribution of rest time among all crew members with precise calculations and customizable options.

Key Features

  • Calculate rest periods with start and end times
  • Support for multiple crew members
  • Configurable gap options (0, 5, 10, 15 minutes)
  • Round to nearest 5 minutes for precision
  • User profiles with role management
  • Save and load personal preferences
  • Modern, intuitive interface
  • Cloud synchronization

How to Use

  1. Set Rest Period: Choose your start and end times
  2. Configure Crew: Select the number of crew members
  3. Choose Gaps: Set gap intervals between breaks (0-15 minutes)
  4. Calculate: View the automatically calculated rest periods
  5. Save Preferences: Store your settings for future use
